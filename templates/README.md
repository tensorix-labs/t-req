# t-req Templates

Starter templates for t-req projects.

## Available Templates

### ship-types
Type-safe API testing with Zod schemas. Implements the "Ship Types, Not Docs" pattern.

**Quick start with degit:**
```bash
npx degit tensorix-labs/t-req/templates/ship-types my-project
cd my-project
bun install
bun test
```

**Or clone and copy:**
```bash
git clone --depth 1 https://github.com/tensorix-labs/t-req
cp -r t-req/templates/ship-types my-project
rm -rf t-req
cd my-project
bun install
```
