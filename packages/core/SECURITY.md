# Security Policy

## Supported Versions

Currently, only the latest version of `@t-req/core` is supported.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

We take the security of this project seriously. If you believe you have found a security vulnerability, please report it privately via [GitHub Security Advisories](https://github.com/tensorix-labs/t-req/security/advisories/new).

Please include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact

We will:
- Acknowledge your report within 48 hours
- Provide an estimated timeline for a fix
- Keep you updated on the progress

Please do not disclose the vulnerability publicly until we have had a chance to address it.
